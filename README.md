# food-store-backend
Backend api for food food ordering system
Project Overview

This is the backend API for a food delivery application that allows:
Customers to browse restaurants, add items to cart, and place orders
Restaurant Owners to manage their menu and handle incoming orders
Transporters, Online Payments, Real-time Tracking
Built with Django + Django REST Framework, designed to serve Flutter(Dart) frontend via a RESTful JSON API.

Landuage         Python
Framework        Django
api              Django-rest-framework
Auth             Jwt
Database         Sqlite(Production) and Postgress Sql(production)
